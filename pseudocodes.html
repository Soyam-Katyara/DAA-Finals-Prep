heapsort.pseudo Heapsort
HEAPSORT(array A): BUILD-MAX-HEAP(A) for i from length(A) down to 2: swap A[1] with A[i] heap_size = heap_size - 1 MAX-HEAPIFY(A, 1) BUILD-MAX-HEAP(A): heap_size = length(A) for i from floor(length(A) / 2) down to 1: MAX-HEAPIFY(A, i) MAX-HEAPIFY(A, i): left = 2i right = 2i + 1 largest = i if left <= heap_size AND A[left] > A[largest]: largest = left if right <= heap_size AND A[right] > A[largest]: largest = right if largest ≠ i: swap A[i] with A[largest] MAX-HEAPIFY(A, largest)
dfs.pseudo Depth-First Search
DFS(graph G, start vertex s): mark s as VISITED process(s) for each neighbor v of s: if v is NOT VISITED: DFS(G, v) // Iterative variant: push s onto Stack while Stack is not empty: v = Stack.pop() if v is NOT VISITED: mark v as VISITED process(v) for each neighbor u of v: if u is NOT VISITED: Stack.push(u)
bfs.pseudo Breadth-First Search
BFS(graph G, start vertex s): mark s as VISITED enqueue s into Queue while Queue is not empty: v = Queue.dequeue() process(v) for each neighbor u of v: if u is NOT VISITED: mark u as VISITED enqueue u into Queue
topo-dfs.pseudo Topological Sort (DFS)
TOPOLOGICAL-SORT-DFS(graph G): for each vertex v in G: mark v as UNVISITED result = empty Stack for each vertex v in G: if v is UNVISITED: DFS-VISIT(G, v, result) return result (top → bottom is topological order) DFS-VISIT(G, v, result): mark v as IN-PROGRESS // detect cycles (gray) for each neighbor u of v: if u is IN-PROGRESS: report cycle (not a DAG) if u is UNVISITED: DFS-VISIT(G, u, result) mark v as DONE // fully processed (black) result.push(v)
topo-bfs.pseudo Topological Sort — Kahn's
TOPOLOGICAL-SORT-BFS(graph G): compute in-degree for every vertex enqueue all vertices with in-degree = 0 into Queue result = [] while Queue is not empty: v = Queue.dequeue() result.append(v) for each neighbor u of v: in-degree[u] = in-degree[u] - 1 if in-degree[u] = 0: enqueue u into Queue if length(result) ≠ number of vertices: report cycle (not a DAG) return result
kruskal.pseudo Kruskal's Algorithm
KRUSKAL(graph G with weighted edges): sort all edges by weight (ascending) initialize a Disjoint-Set (Union-Find) with each vertex in its own set MST = [] for each edge (u, v, weight) in sorted order: if FIND(u) ≠ FIND(v): // u and v in different components MST.append((u, v, weight)) UNION(u, v) if length(MST) = |V| - 1: break // MST is complete return MST
dijkstra.pseudo Dijkstra's Algorithm
DIJKSTRA(graph G, source s): dist[s] = 0 dist[v] = ∞ for all other vertices v prev[v] = UNDEFINED for all v enqueue all vertices into MinPriorityQueue keyed by dist while PriorityQueue is not empty: u = vertex with minimum dist (extract-min) for each neighbor v of u: alt = dist[u] + weight(u, v) if alt < dist[v]: dist[v] = alt prev[v] = u decrease-key of v in PriorityQueue return dist[], prev[]
bellman-ford.pseudo Bellman-Ford Algorithm
BELLMAN-FORD(graph G, source s): dist[s] = 0 dist[v] = ∞ for all other vertices v prev[v] = UNDEFINED for all v // Relax all edges |V| - 1 times repeat |V| - 1 times: for each edge (u, v, weight) in G: if dist[u] + weight < dist[v]: dist[v] = dist[u] + weight prev[v] = u // Detect negative-weight cycles for each edge (u, v, weight) in G: if dist[u] + weight < dist[v]: report "negative-weight cycle detected" return dist[], prev[]
