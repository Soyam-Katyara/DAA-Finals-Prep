What is a Heap?

A heap is a complete binary tree where each node satisfies the heap property relative to its children. It is efficiently stored as a flat array — no pointers needed.

Min Heap
Parent ≤ Children
Root = smallest element
Max Heap
Parent ≥ Children
Root = largest element

Array Index Formulas (1-based)

Left Child → 2 × i
Right Child → 2 × i + 1
Parent → ⌊ i / 2 ⌋

Time Complexity

  • Insert (heapify up) → O(log n)
  • Extract Root (heapify down) → O(log n)
  • Peek Root → O(1)
  • Heap Sort (build + extract all) → O(n log n)

Common Use Cases

  • Priority Queues — OS scheduling, Dijkstra, A* pathfinding
  • Heap Sort — in-place, O(n log n) comparison sort
  • Finding k-th largest/smallest in a stream
  • Median maintenance in real-time data
